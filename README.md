# 🥁 Drum Kit Web App

A fun and interactive **Drum Kit Web Application** built using **HTML, CSS, and JavaScript**.  
Play different drum sounds using your **keyboard** or **mouse clicks**.

---

## 🚀 Features

- 🎵 Play sounds using keys: `q, w, e, r, t, y`
- 🖱️ Click buttons to trigger sounds
- ⚡ Instant sound response
- 🎨 Clean UI with simple animations
- 🔊 Supports both lowercase & uppercase keys (different sounds)

## 🎹 Controls

Key             :	  Action
- q, w, e, r, t, y	: Drum sounds
- Q, W, E, R, T, Y	: Special sounds
- Mouse Click	    : Play sound + animation

## ⚙️ How It Works

- Detects button clicks and key presses
- Plays audio using JavaScript Audio object
- Adds animation using CSS class .pressed

## 🎨 UI Features
- Gradient background
- Circular drum buttons
- Shadow effects
- Click animation
